CSS 고급 선택자

하위 요소 선택자

특정 하위 요소의 후손 요소만을 선택한다. 뜨어쓰기를 통해 구분한다.

자식 선택자

후손이 아닌 바로 밑에 자식요소만을 선택한다.
기호 > 를 사용한다.

인접 형제 선택자

특정 요소 바로 뒤에오는 형제 요소 1개만을 선택한다.

일반 형제 선택자

특정 요소 바로 뒤에오는 모든 형제 요소를 선택한다.

가상 클래스

사용자가 특정 행동을 했을 때 가상의 클래스가 붙은 것처럼 동작한다.


"선택자 :가상클래스" 의 형태로 사용한다.

가상 요소

HTML 문서상의 작성되지 않았지만, 디자인적으로 요소가 필요할 때 가상으로 만들어 낸다.
'선택자 :: 가상요소' 의 형태로 사용한다.
