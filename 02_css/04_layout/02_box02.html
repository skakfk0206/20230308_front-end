CSS 여백과 테두리

border

border는 박스의 테두리의 두께, 졸류, 색상을 지정한다.
border를 지정하면 요소의 너비에 영향을 미친다.

border 개별속성

  • border-width: 테두리 선의 두께를 지정한다.
  • border-style:테두리 선의 종류를 지정한다.
  • border-color: 테두리 선의 색상을 지정한다.
  • border-radius: 모서리를 둥굴게 처리한다.

margin

margin은 요소의 바깥 여백을 지정한다.
margin을 지정해도 요소의 너비에 영향을 미치지 않는다.

padding

padding은 요소 안쪽여백(컨텐츠content 와 보더border 사이)를 지정한다. padding을 지정하면 너비와 높이에 영향을 미친다.

box-sizing

요소의 너비와 높이 계산 방법을 지정한다.

  • content-box(기본값) : 콘텐츠 영역의 너비와 높이를 먼저 지정하고, 나머지를 계산한다.
  • boeder-box: boeder영역을 포함해서 너비와 높이를
