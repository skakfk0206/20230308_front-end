CSS float

float 속성의 방향을 right,left로 지정하면 요소가 떠있는 상태가 된다.
float된 요소바로 다음 요소는 그밑에 위치하게 된다.
컨텐츠(인라인)들은 위로 올라오는 float되는것을 허용하지 않는다.

clear라는 속성을 통해 float을 해제할수 있다
float된 요소의 다음 요소에 사용한다.
