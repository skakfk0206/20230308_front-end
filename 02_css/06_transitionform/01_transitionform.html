CSS transform

요소를 변형시킬 수 있다.
transform의 값은 변형 함수를 사용한다.

translate(이동)

요소를 현재 위치를 기준으로 움직인다.

X
Y
XY
Z

scale(크기)

요소의 크기를 변형한다.
요소 안의 컨텐츠도 같이 변형된다.

X
Y
XY

rotate(회전)

요소를 회전시킨다.
deg(각도) 단위를 사용한다.

X
Y
Z

skew(왜곡)

요소를 비틀어서 왜곡시킨다.
deg(각도) 단위를 사용한다.

X
Y
Z

transform-origin

변형이 일어나는 기준 축을 바꿀 수 있다.
